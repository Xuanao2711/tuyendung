      \nX\u00e2y d\u1ef1ng v\u00e0 th\u1ef1c hi\u1ec7n c\u00f4ng t\u00e1c d\u1ef1 b\u00e1o ng\u1eafn v\u00e0 trung h\u1ea1n, d\u1ef1a tr\u00ean k\u1ebft qu\u1ea3 theo d\u00f5i s\u1ed1 li\u1ec7u v\u00e0 c\u00e1c \u00fd ki\u1ebfn ph\u00e2n t\u00edch<\/li>\n
